Instructions

Welcome to the Simon Game! Here's how to play:

  1. Press any key to start the game.
  2. Watch the pattern of colors and remember it.
  3. Click on the buttons in the same order as the pattern.
  4. If you get the pattern right, the level increases.
  5. If you make a mistake, the game is over.
  6. Have fun and challenge yourself to reach higher levels!
